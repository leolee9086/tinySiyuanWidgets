部署设置

提示：Token通常位于思源笔记设置 - 关于中的API TOKEN部分。

⚠️ 安全警告：API Token 拥有操作您整个笔记库的权限！请仅在您完全信任的页面（例如本地运行的此页面）或应用中输入。切勿在来源不明的网站或应用中泄露您的 Token！

💡 **更安全的建议**：为了避免在此页面直接暴露您的主 API Token，可以考虑采用"**临时 Token 模式**"。即在需要执行部署操作前，手动在思源设置中生成一个临时的 API Token，用这个临时 Token 在此页面进行部署。部署完成后，回到思源设置中将 Token 修改回您常用的主 Token，或删除该临时 Token。虽然步骤稍繁琐，但这可以有效提高安全性。

准备就绪，请选择要部署的挂件。
